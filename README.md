# Java 21 JRE

Installs Java21 JRE on a Debian-based Linux host.

## Requirements

Debian-based Linux.

## Role Variables

None.

## Dependencies

None.

## Example Playbook

```
    - hosts: servers
      roles:
         - { role: freeradical604.ansible-java21 }
```
